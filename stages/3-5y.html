🛡️ 装备清单 ​

安全必备 ​

🚗 安全座椅（前向/增高垫过渡） Essential ​

体重身高达标前持续使用。从后向转前向，再到增高垫，别急着"毕业"。（AAP/CPSC）

选购 沿用 0-3M 阶段购买的座椅，或根据体重升级增高垫：Cybex Solution / Joie Trillo

⛑️ 运动护具（头盔/护膝） Essential ​

自行车、滑板车阶段，头部外伤预防是第一位的。从第一次骑就戴上。（CDC）

选购 micro / Globber / 迪卡侬，¥50-200/套

👁️ 定期视力筛查 Essential ​

3-5岁是弱视治疗的窗口期，错过不可逆。每年至少一次专业筛查。（AAP）

选购 当地三甲医院眼科 / 专业儿童眼科诊所，每年1次

兴趣启蒙 ​

🏊 结构化运动课（游泳/体操） Conditional ​

4岁以上可以开始系统学习运动技能。选孩子感兴趣的，不必追求"全面发展"。

选购 本地口碑机构，建议先上体验课。游泳/体操/跆拳道/轮滑均可，¥100-300/课

🎹 乐器启蒙 Conditional ​

音乐训练与执行功能有相关性，但核心驱动力应该是兴趣，不是"开发大脑"。

选购 从体验课开始，钢琴/尤克里里/打击乐入门。琴行租琴试学，确认兴趣再买

❌ 避坑清单

  • 防蓝光眼镜（无近视） — 没有证据蓝光导致儿童近视。近视预防的核心是户外时长（每天2h），不是防蓝光。（AAO）
  • "全脑开发"/感统训练（无诊断） — 无医学诊断的感统训练无RCT。替代：自由玩耍 + 户外 + 桌游

📊 全阶段发展算法速查卡 ​

0-3M  趴 + 看脸 + 听说话 + 被抱
3-6M  翻 + 抓 + 回应对话 + 摸
6-12M 爬 + 躲猫猫 + 指认 + 手指食物
1-3Y  读 + 跑 + 涂 + 说情绪 + 上厕所
3-5Y  桌游 + 运动 + 户外2h + 为什么 + 和小朋友玩

回顾全程 🎯

每个阶段的"算法"都指向同一件事：不是加速发展，而是不错过窗口期。大自然已经写好了程序，你只需要提供环境和陪伴。
