# 👶 0-3M · 感官校准期

> 这个世界好大，而你的声音好近。
>
> 趴着、看脸、听说话、被抱着 —— 这就是这个阶段的全部"课程"。

## 🎯 发展里程碑

| 维度 | 科学项目 | 在做什么 | 每天多久 | 来源 |

| 视觉 | **黑白卡**（20-30cm） | 新生儿视力仅20/400，高对比度促进视神经发育 | 每次2-5min | AAP |

| 依附 | **肌肤接触** Skin-to-Skin | 稳定心率体温 + 催产素双向分泌 → 安全依附 | 每天≥1h | WHO |

这个阶段不需要任何课程、教具或App。只需要：趴着、看脸、听说话、被抱着。
:::

::: danger ❌ 避坑清单
- **婴儿床围 Crib Bumpers** — CPSC 2022立法禁售，窒息致死因果链明确
- **定型枕** — AAP禁止1岁内任何枕头（SIDS风险），扁头多数自愈，Tummy Time是最好的预防
- **游泳脖圈** — 颈椎压迫 + 滑脱窒息风险，零发育收益。替代：大人一对一水中托抱
- **恒温调奶器** — 保温壶 + 温度计完全够用
- **婴儿专用洗衣液** — 普通无香型洗衣液就行
:::
