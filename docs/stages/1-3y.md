# 🗣️ 1-3Y · 语言爆发期

> 从 mama 到"妈妈我不要这个我要那个为什么为什么为什么"。
>
> 语言像瀑布一样涌出来。户外跑跑跑，绘本翻翻翻，泥巴捏捏捏。

---

## 🎯 发展里程碑

| 维度 | 科学项目 | 在做什么 | 每天多久 | 来源 |
|------|---------|---------|---------|------|
| 语言 | **亲子共读 + 延展对话** | 每天15min共读 → 3岁词汇量差3倍 | 15-30min/天 | AAP |
| 大运动 | **户外无动力探索** | 攀爬/跑/跳 = 前庭+本体感觉自然整合 | 每天≥1h户外 | AAP |
| 精细运动 | **涂鸦 / 黏土 / 撕纸** | 手指小肌群精细控制 → 为握笔书写做准备 | 融入日常玩耍 | — |
| 社会情感 | **情绪命名** Emotion Labeling | "你现在很生气" → 前额叶调节杏仁核回路建立 | 情绪发生时即时 | Lieberman |
| 自主性 | **如厕训练**（18-24M窗口） | 括约肌成熟 + 模仿意愿 = 窗口期 | 跟随孩子信号 | AAP |

::: warning 科学边界 🔬
1-3Y是"感统训练"营销重灾区。无医学诊断的感统训练没有RCT支持。真正的感觉统合在户外自由玩耍中就能自然完成 —— 踩水坑、爬土坡、滚草地，比任何课程都有效。
:::

## 🛡️ 装备清单

### 安全必备

#### 🦷 含氟牙膏 + 儿童牙刷 Essential

出牙即开始刷，米粒大小含氟牙膏，降低龋齿率。龋齿是可以预防的慢性病。（AAPD）

**选购** 狮王(Lion) / 巧虎 / 舒客宝贝，¥20-50

#### ☀️ 防晒霜 SPF30+ Essential

儿童期紫外线累积与成年黑色素瘤有明确因果链。户外刚需，全年使用。（AAP）

**选购** Thinkbaby / 安热沙儿童版 / Babyganics，¥50-150

#### 🚲 平衡车 / 三轮车 + 头盔 Conditional

促进前庭系统和大运动发展。**头盔必配**，从第一次骑就养成习惯。（AAP）

**选购** 平衡车 Strider / KinderKraft ¥300-800 + 头盔 micro/迪卡侬 ¥100-200

### 生活装备

#### 🚽 小马桶 / 如厕训练座 Essential

18-24个月是如厕训练窗口期。矮小的、脚能着地的小马桶，比大人马桶上的辅助圈好用。

**选购** BabyBjörn / Fisher-Price / 可优比，¥80-300

### 认知与情感

#### 📚 绘本 Board Books Essential

出生就可以开始亲子共读，每天15分钟效果量巨大。好绘本不贵，图书馆也是好选择。（AAP）

**选购** 图书馆借阅 + 京东/当当自营囤货，¥0-500/年

#### 🧱 开放式玩具（积木/黏土/画笔） Essential

发散性思维 + 精细运动，远优于电子声光玩具。（AAP）

**选购**

| 推荐 | 品牌 | 亮点 | 入手价 |
|------|------|------|--------|
| 传世之选 | Grimm's | 手工椴木 + 无毒水性漆 + 残值70%+ + 用10年 | ≤¥800（团购） |
| 向上兼容 | LEGO DUPLO | 兼容标准LEGO + 残值稳定 + ABS安全标准最严 | ≤¥260（拼多多百补） |
| 环保之选 | Plan Toys | 橡胶木回收料 + B Corp + Red Dot | — |
| 国产优选 | 木玩世家 | 云和产业带龙头（全球60%木玩产自浙江云和） | Plan Toys 20-30%价格 |

::: danger ❌ 避坑清单
- **早教机/点读笔（高端款）** — 2岁前零屏幕，2-5岁≤1h/天。人际互动效果远大于设备。替代：真人共读
- **"右脑开发"闪卡** — "右脑开发"这个概念本身就没有RCT支持。替代：对话 + 绘本
- **感统训练课（无诊断）** — 无医学诊断的感统训练无RCT。替代：自由玩耍 + 户外
:::
