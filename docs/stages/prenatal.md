# 🤰 孕期准备

> 从两条杠到产房的那些日子 —— 科学筹备，给自己一个安心的开始。

---

## 🎯 这个阶段的关键任务

| 优先级 | 任务 | 为什么重要 | 时间节点 |
|--------|------|-----------|---------|
| ⭐⭐⭐ | 叶酸 400μg/日 | 降低神经管缺陷风险70% | 孕前3月启动 |
| ⭐⭐⭐ | 规范产检 | 筛查妊娠糖尿病/子痫前期/胎儿异常 | 全孕期 |
| ⭐⭐ | 铁剂/复合维生素 | 贫血孕妇补铁降低早产率（按医嘱） | 按需 |
| ⭐⭐ | 研究安全座椅+婴儿床 | 高单价品，锚定大促可省 ¥1,000+ | 距预产期6个月 |

- **防辐射孕妇服** — 日常电磁辐射远低于致害阈值，无RCT支持，纯智商税
- **家用胎心仪** — NHS/RCOG明确反对：假阴性可能延误就医。替代：规范产检 + 数胎动
- **胎教音箱/课程** — 胎儿30周才发育听觉，你日常说话就是最好的"胎教"
:::
