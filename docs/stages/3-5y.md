# 🧩 3-5Y · 执行功能期

> "爸爸我赢了！""你耍赖！""我们再来一局！"
>
> 桌游、运动、户外两小时、和小朋友的江湖 —— 一个小小思想家正在诞生。

---

## 🎯 发展里程碑

| 维度 | 科学项目 | 在做什么 | 每天多久 | 来源 |
|------|---------|---------|---------|------|
| 执行功能 | **规则类桌游 / 假装游戏** | 等轮次(抑制控制) + 记规则(工作记忆) + 换策略(认知灵活性) | 20-30min/天 | Diamond |
| 大运动 | **结构化运动** | 游泳/体操/跳绳 → 系统运动技能 + 心肺发育 | 每周2-3次 | AAP |
| 近视预防 | **户外光照暴露** | 多巴胺介导巩膜生长抑制 → 户外是近视预防的唯一强因果因素 | 每天≥2h | AAO |
| 语言/认知 | **开放式提问** | "你觉得为什么？" → 推理 + 叙事能力 | 融入日常 | — |
| 社会化 | **同伴自由游戏** | 合作、协商、冲突解决的真实训练场 | 每周多次 | — |

## 📊 全阶段发展算法速查卡

```
0-3M  趴 + 看脸 + 听说话 + 被抱
3-6M  翻 + 抓 + 回应对话 + 摸
6-12M 爬 + 躲猫猫 + 指认 + 手指食物
1-3Y  读 + 跑 + 涂 + 说情绪 + 上厕所
3-5Y  桌游 + 运动 + 户外2h + 为什么 + 和小朋友玩
```

::: tip 回顾全程 🎯
每个阶段的"算法"都指向同一件事：**不是加速发展，而是不错过窗口期**。大自然已经写好了程序，你只需要提供环境和陪伴。
:::
