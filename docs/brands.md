## 综合策略：溢价值不值得？

| 品类 | 策略 | 理由 |
|------|------|------|
| 安全座椅 | 买进口 | 认证深度 + 残值覆盖溢价 |
| 婴儿床 | 分化 | 追求残值 → Stokke；纯功能 → IKEA |
| 餐椅 | 买Tripp Trapp | 终身使用，年均趋近于零 |
| 睡袋 | 买国产 | 高频替换消耗品，进口溢价无残值回收 |
| 吸奶器 | 买Spectra | 韩国供应链 = "进口品质 + 国产价格" |
| 玩具 | 核心Grimm's + 日常国产 | 经典款锚定 + 消耗用对冲 |
| 防晒/口腔 | 买国产 | 消耗品，国产供应链成熟 |
