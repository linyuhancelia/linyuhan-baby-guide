# 阅读指南

## 这本手册是什么？

这是一份基于科学文献的育儿物品清单和发展指南，覆盖孕期到5岁。

我们做了三件事：

1. 用 AAP/WHO/CDC/NHS 的循证指南判断每件东西**该不该买**
2. 给"该买"的东西匹配**好看又值得**的品牌
3. 告诉你**什么时候买最划算**

然后把这三层信息融合到每个成长阶段里，沿着时间线一路往前走就好。

## 判定标准

| 标记 | 含义 | 准入条件 |
|------|------|---------|
| Essential | 科学必选 | 权威指南明确推荐 + 因果证据（RCT/大样本/法规） |
| Conditional | 按需配置 | 合理机制但非刚需，因个体差异决定 |
| Cognitive-Tax | 智商税 | 权威机构反对 / 无RCT / 可低成本替代 |

::: tip 三个标记怎么用？
- **Essential** 是底线清单 —— 预算有限时只买这些就够了
- **Conditional** 看你的具体情况 —— 每个条目都标注了决策条件
- **Cognitive-Tax** 建议跳过 —— 都标注了低成本替代方案
:::

## 关于品牌审美

我们的审美基准：**北欧极简 / 侘寂风 / 中性色系**

预算基准：追求长期持有价值与安全性，像管理优质固收策略一样管理母婴消费。

## 关于数据来源

- **科学证据**：AAP（美国儿科学会）、WHO（世界卫生组织）、CDC（美国疾控中心）、NHS（英国国民健康服务）、CPSC（美国消费品安全委员会）
- **价格数据**：2024-2026年天猫/京东/拼多多公开大促记录
- **品牌评估**：安全认证 × 设计奖项 × 二手残值 × 用户口碑
